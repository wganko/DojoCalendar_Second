# DojoCalendar - Second Stage

GAS (Google Apps Script) アプリケーション for DojoCalendar Second Stage.

## 概要

- **GitHub フロント**（LIFF）: userId/displayName 取得 → Apps Script API に送信
- **Apps Script バック**: スプレッドシート管理、ICS 配信、アクセス履歴記録

## セットアップ

### 1. clasp のインストール

```bash
npm install
```

### 2. 認証

```bash
npx clasp login
```

### 3. コードをプッシュ

```bash
npm run push
```
